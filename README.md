# learning_contents
出張インターンの講義で行うコンテンツ

これはCRUDを体感で学ぶためのツールです。

## 作成目的
Webアプリケーションの基本であるCRUD処理を学習するために作成しました。

## 主な機能
- データの新規作成（Create）
- データ一覧の表示（Read）
- データ内容の編集（Update）
- データの削除（Delete）

## 使用技術
- フロントエンド：HTML, CSS
- バックエンド：Google Apps Script
- データベース（模倣）：Google spreadsheet
- 開発環境：VScode
- その他：github
  
## 画面構成
- 一覧画面
- 新規作成画面
- 詳細画面
- 編集画面

## 手順

①セットアップ
まずはURLの中に入り、gitのページにいく

codeタブから、zipファイルをダウンロードする

デスクトップ上で解凍する

VSコード上でフォルダを開く

とりあえずのセットは完了




②続いてスプレッドシートと持ってきてもらったコードを繋げます

Gドライブを開く

ダウンロードしてもらったフォルダの中からk-sインターンコンテンツ制作モデル.xlsxをGドライブ上でドラック＆ドロップでアップロードする

アップロードができたらspreadシートで開く

「ファイル」タブからスプレッドシートとして保存

VSコード上のgas/code.js のコードを全コピー（ctrl+A,ctrl+C)する

スプレッドシートに戻り、拡張機能タブからAPPS SCRIPTを開く

ウィンドウの右側（コードを書くスペース）で全選択から全ペースト（ctrl+A,ctrl+V)

右上のデプロイタブから新しいデプロイ

種類の選択→ウェブアプリ「アクセスできるユーザー」を全員にして、デプロイ

発行されたURLをコピーし、VSコード上にあるscript.jsへ移動する

設定上部にある、API URLを選択してペーストする

ctrl+Sで保存する

HTMLに移動して右クリックからLive severを開く
