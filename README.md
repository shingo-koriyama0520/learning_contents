# learning_contents
出張インターンの講義で行うコンテンツ

これはCRUDを体感で学ぶためのツールです。

## 作成目的
Webアプリケーションの基本であるCRUD処理を学習するために作成しました。

## 主な機能
- データの新規作成（Create）
- データ一覧の表示（Read）
- データ内容の編集（Update）
- データの削除（Delete）

## 使用技術
- フロントエンド：HTML, CSS
- バックエンド：Google Apps Script
- データベース（模倣）：Google spreadsheet
- 開発環境：VScode
- その他：github
  
## 画面構成
- 一覧画面
- 新規作成画面
- 詳細画面
- 編集画面
